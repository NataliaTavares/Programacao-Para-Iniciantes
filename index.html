HTML - Hypertext Markup Language ou traduzido para o português Linguagem de Marcação de Hipertexto

Html é uma linguagem de marcação utilizada para a construções de páginas web, ela é composta principalmente por tags e informações como textos, imagens, vídeos etc.

As tags são conhecidas por se encontrarem entre os sinais de maior e menor.

Sobre as Tags

  • Todas as tags devem ser escritas em letras minúsculas
  • Tags que possuem uma de abertura e outra de fechamento, é obrigatório o fechamento (abriu tag -> fechou tag).
  • Tags que não possuem outra para fechamento (elementos vazios) devem ser fechadas por si só.

Estrutura básica de um documento HTML

Nessa estrutura temos os elementos básicos:

<html> </html> = Indica ao browser o início de documento html. Ele deve ficar no início do arquivo abaixo do Doctype e seu respectivo fechamento no final. Todo arquivo HTML deve conter essa tag e todo conteúdo da página criada deve ficar dentro dela.

<head> </head> = São definidos os atributos gerais á página como metadados por exemplo.

<title> </title> = Título da janela , fica posicionado no head.

<body> </body> = Seção de conteúdo da página. Aqui devera ficar todo o corpo da sua página ou seja , toda apresentação como texto, imagens e etc.

<!DOCTYPE> = Se trata de uma declaração que informa ao navegador qual é a versão do HTML que está sendo utilizada no arquivo.
OBS: Ele deve ser é primeira declaração em um documento web.

Outras tags do HTML

<h1>,<h2>,<h3>,<h4>,<h5> e <h6> = Tags para definir um título e subtítulos, onde h1 é o título maior e mais importante, até o h6 que é menor e menos importante;

<p> = Tag para definir um parágrafo;

<a> = Elemento para criar uma conexão “link” com outros recursos da web;

<header> = Define um cabeçalho;

<section> = Define uma seção;

<article> = Define um artigo;

<div> = Define uma divisão;

<footer> = Define um rodapé;

<nav> = Define uma área de navegação (como menus);

<table> = Define uma tabela;

<ol> = Define uma lista ordenada;

<ul> = Define uma lista não ordenada;

<li> = Define o item de uma lista;

<form> = Define um formulário;

<input> = Define os campos do formulário;

<textarea> = – Insere uma área para o usuário digitar um texto;

<button> = Cria um botão;

<img> = Permite inserir uma imagem;

<br> = Permite pular linha;

<!--> = Permite inserir um comentário;
